# İnsan Kaynakları Yönetimi Platformu IKPLUS – Tamamlanmış Bireysel Versiyon  

Bu proje, başlangıçta bir grup çalışması olarak başladı ancak daha sonra bireysel olarak tamamlandı.  
Eksik özellikler tamamlandı, performans iyileştirmeleri yapıldı ve yeni özellikler eklendi.

## 📌 Proje Durumu  
✅ Grup çalışmasıyla belirli bir aşamaya kadar geliştirildi.  
✅ Eksik özellikler tamamlandı ve kod iyileştirildi.  
🚀 Yeni özellikler eklendi ve geliştirme süreci devam ediyor.

## 🛠 Kullanılan Teknolojiler  
- **Backend:** Spring Boot  
- **Frontend:** React + TypeScript  
- **Veritabanı:** PostgreSQL  
- **Auth:** JWT  Spring Security

## 📂 Proje Yapısı  
📁 `backend/` – Spring Boot ile geliştirilen backend  
📁 `frontend/` – React + TypeScript ile geliştirilen frontend  

> 📌 **Not:** Bu versiyon, proje ekibiyle belirli bir aşamaya kadar geliştirilen [Grup Çalışması Versiyonundan](https://github.com/Shatice92/IKPLUS.git) ayrılarak bireysel olarak tamamlanan ve genişletilen versiyondur.
